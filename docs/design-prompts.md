```
Design a mobile app (iOS & Android) called "Kuya Bong" — a booking and clinic-management
app for a well-known physiotherapist & chiropractor who runs two clinics (Clinic A and
Clinic B). The app serves two roles in one app: Patient and Administrator (Kuya Bong).

Core patient screens to design:
1. Welcome / onboarding (login + register)
2. Patient Home dashboard — upcoming appointment, clinic reminder, package balance, "Book" shortcut
3. Book Appointment flow — select clinic → calendar (available dates) → time slots → review → confirmation
4. My Appointments — tabs for upcoming / completed / cancelled / rescheduled
5. My Packages — sessions remaining, expiry date, family usage history
6. Family — link spouse, add child
7. Profile

Core admin screens to design:
1. Admin Dashboard — today's appointments, pending requests, follow-up reminders
2. Availability Calendar — publish/edit appointment slots by clinic, date, time
3. Appointments management — filter by clinic / date / status
4. Patient profile view
5. Packages — create package, assign to patient, usage tracking
6. Products — herbal & supplement catalogue with prices, record purchase, follow-up list

Design intent: trustworthy, calm, health & wellness feel; very easy booking in few taps;
clear distinction between Clinic A and Clinic B; large readable typography for older patients;
accessible contrast; rounded cards; bottom tab navigation for patient, side/dashboard layout
for admin.

Pick the single best visual style for a health & wellness clinic booking app from these three
directions, and apply it consistently across all screens:
- Option 1 "Calm Clinical": teal/green + white, soft, medical-trust, minimalist.
- Option 2 "Warm Wellness": terracotta/sage + cream, friendly spa-like, soft shadows.
- Option 3 "Modern Bold": deep navy + bright accent, high-contrast, confident, data-forward.

Choose the option that maximizes trust, readability for older users, and ease of booking,
then generate the full screen set in that style.

Visual style: "Calm Clinical". Primary color teal/medical green (#0E7C7B) with plenty of
white space. Clean, minimalist, professional. Inter or SF Pro typography, large and legible.
Soft rounded cards, thin outlines, subtle shadows, simple line icons. Mood: trustworthy,
clinical, calm. Generous padding, accessible contrast, friendly for older patients.
```
